More Shell Things

Special path aliases

Try

							
								# Print working directory
								pwd
								# List some directories
								ls -la ~
								ls -la ~admin
								ls -la .
								ls -la ..
								# Change working directory
								cd /share/datasets
								# List some more
								ls -la .
								ls -la ..

Environment variables

  • Store essential information
  • Can be used interactively and in scripts
  • $PWD - Your current working directory
  • $USER - Your username
  • $HOME - Your home directory
  • $PATH - A list of directories where Linux will look for programs

Output redirection

stdout and stderr

  • "Normal" program output goes to stdout
  • Log and error messages go to stderr
  • We can redirect that output into a file using the > operator
							
								# Check who is online
								who
								# Save who is online into a file
								who > ~/online_users.txt
								# Read that file back
								less ~/online_users.txt

Redirecting stdout to programs using the |

  • Output of program on the left becomes input of program on the right
							
								tail -n +1 /share/datasets/pokemon.tsv |
								sort -t $'\t' -k5nr |
								head
