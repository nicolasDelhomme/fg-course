The shell

The shell is the software on the computer that interprets the commands

Shell timeline

  • The Bourne shell: 1977 
    sh
  • The Bourne-again shell: 1989 
    bash
  • The Z shell: 1990 
    zsh

Bash like shells (Bash & Zsh)

  • Auto completion
  • History
  • Advanced editing
  • Pattern matching, brace expansion

A shell command

Let's look at a command: Coreutils' rm to remove files and directories

							
								rm -r -f --no-preserve-root --interactive=once /

Getting Help

							<prog> -h

When in doubt... Google (DuckDuckGo)

SSH: Secure Shell

  • Not a shell
  • Communication protocol to securely connect to a remote computer and send commands to it

Example command

							
								ssh -p <PORT> <user>@<host>
								ssh -p 922 bastian@microasp.upsc.se
