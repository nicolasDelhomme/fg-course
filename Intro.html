What is Linux

Linux is a Kernel.

The kernel handles communication between hardware and software.

Linux, Unix, BSD... are different implementations of the same idea

What are we using in a typical Linux-based OS?

  • Linux as a kernel
  • GNU coreutils as apps on top of the kernel
  • Custom software on top of that
  • [A Desktop environment]

Linux distributions

  • ...are packages of a kernel an customized software on top
  • ...tend to have a particular focus

Debian Ubuntu Elementary Mint

  • Easy to use
  • Stable
  • Beginner friendly
  • Software several updates behind
  • Bulky & Resource heavy
  • Less customizable

Arch Gentoo

  • Very customizable
  • Bleeding edge
  • High performance
  • Rolling release model
  • Needs a lot of know-how to use well
  • Even then, tedious to use

MX Linux Manjaro

  • Best of both worlds
  • Active community & development
  • Young
  • Can have stability issues

Red Hat Fedora CentOS

  • Business facing
  • Current, well maintened
  • Secure
  • Not free (Red Hat)
  • Overkill for desktop (CentOS)
  • Too many updates?
